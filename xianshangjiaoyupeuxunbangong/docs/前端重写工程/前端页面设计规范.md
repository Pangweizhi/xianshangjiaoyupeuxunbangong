# 前端页面设计规范

## 1. 目标

这份文档统一前台和后台页面的视觉与结构标准，避免同一套系统出现多套风格、多个模板、多个交互习惯。

## 2. 页面设计总原则

- 统一
- 简洁
- 层次清楚
- 可扩展
- 现代化
- 易维护

## 3. 前台页面规范

### 3.1 适用范围

前台页面面向学生和教师，重点是内容展示和用户参与。

### 3.2 页面结构

- 首页
- 登录 / 注册
- 内容列表
- 内容详情
- 个人中心
- 互动页面

### 3.3 风格要求

- 更有品牌感
- 更大留白
- 更强的视觉焦点
- 卡片化展示
- 适度使用渐变和插画

### 3.4 常见页面模板

#### 首页模板

- 顶部导航
- 主视觉区
- 推荐内容区
- 快捷入口区
- 信息公告区
- 页脚区

#### 列表模板

- 页面标题
- 简短说明
- 搜索筛选区
- 内容列表区
- 分页区

#### 详情模板

- 标题区
- 元信息区
- 封面区
- 内容区
- 附件区
- 相关推荐区

#### 个人中心模板

- 左侧导航
- 右侧内容区
- 顶部个人卡片
- 底部业务内容区

## 4. 后台页面规范

### 4.1 适用范围

后台页面面向管理员，重点是效率、清晰和稳定。

### 4.2 页面结构

- 登录页
- 首页仪表盘
- 列表管理页
- 新增编辑页
- 详情页
- 系统工具页

### 4.3 风格要求

- 简洁专业
- 少装饰
- 高信息密度
- 强操作导向
- 统一表格、表单、弹窗风格

### 4.4 常见页面模板

#### 列表模板

- 页面标题
- 搜索区
- 操作区
- 表格区
- 分页区

#### 表单模板

- 标题
- 说明
- 表单项
- 校验提示
- 提交按钮

#### 详情模板

- 标题
- 基础信息
- 业务内容
- 附件/图片
- 返回入口

#### 仪表盘模板

- 统计卡片
- 图表区
- 待办区
- 快捷入口区

## 5. 通用视觉要求

- 标题层级清楚
- 文本对比足够
- 卡片圆角统一
- 间距统一
- 阴影克制
- 主色只用于重点强调

## 6. 响应式要求

- 桌面端优先
- 移动端可用
- 列表页和详情页在小屏幕下自动调整为单列或折叠布局

## 7. 交付标准

页面设计规范完成后，至少应满足：

- 前台与后台风格统一但气质不同
- 核心页面能直接套用模板
- 新页面能快速复用已有结构
- 视觉层统一，后续扩展不乱
