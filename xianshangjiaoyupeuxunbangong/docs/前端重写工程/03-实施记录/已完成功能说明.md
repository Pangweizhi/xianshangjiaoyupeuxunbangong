# 已完成功能说明

## 1. 说明

这份文档汇总当前“新前端重写工程”已经完成的内容，目的是让后续查看的人快速知道：

- 已经做到了哪里
- 哪些功能已经不是页面空壳
- 哪些功能已经接入真实后端

## 2. 已完成的工程基础

已经在仓库根目录新建独立工作区 `new-frontend`，并拆分为三个子工程：

- `front-web`
- `admin-web`
- `shared`

已完成的基础能力包括：

- Vue 3 + Vite 新工程结构
- 前后台分离
- 共享 HTTP 请求层
- Token 注入
- 登录态持久化
- `withCredentials` 兼容旧后端 `session`
- 共享类型定义

## 5. 作业提交流程已完成内容

这是当前重写里最关键的一条业务链，已经完成：

1. 学生前台进入作业详情
2. 上传附件
3. 提交到后端 `zuoye_submit`
4. 后台提交记录页查看记录
5. 教师批改
6. 教师评分
7. 教师填写评语
8. 教师退回重交
9. 前台查看最近一次提交结果

## 6. 后端新增内容

为了支撑作业提交和批改，已经新增后端模块 `zuoye_submit`。

主要内容包括：

- 新表：`zuoye_submit`
- 新实体
- 新控制器
- 新 Mapper
- 新 SQL

当前已支持的字段和业务含义：

- `submitStatus`：提交状态
- `submitScore`：评分
- `submitRemark`：评语
- `checkTime`：批改时间

## 10. 当前限制

当前环境仍然存在一个明确限制：

- `pnpm` 不存在
- `node` 和 `npm` 执行返回“拒绝访问”

因此当前能确认的是：

- 源码已经落地
- 结构已经整理
- 文档已经补齐
- 逻辑已经按现有系统接口接上

但还不能在这台机器上完成真实启动和联调验证。
