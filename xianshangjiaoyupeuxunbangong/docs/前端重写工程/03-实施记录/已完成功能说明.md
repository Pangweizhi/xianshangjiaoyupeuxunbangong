# 已完成功能说明

## 1. 文档目的

这份文档用于说明当前仓库里已经落地的功能，方便后续继续开发时快速判断：

- 哪些功能已经不是空壳
- 哪些页面已经接上真实接口
- 哪些后端能力已经写入系统

## 2. 新前端已完成内容

当前 `new-frontend` 已完成的基础结构：

- `front-web`
- `admin-web`
- `shared`

已具备的通用能力：

- Vue 3 + Vite 工程结构
- 共享 HTTP 请求层
- Token 注入
- 登录态持久化
- `withCredentials` 兼容旧后端 `session`
- 公共类型定义
