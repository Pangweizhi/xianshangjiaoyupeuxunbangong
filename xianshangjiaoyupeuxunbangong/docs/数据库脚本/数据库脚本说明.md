# 数据库脚本说明

## 1. 当前只保留一份脚本

作业提交相关数据库脚本统一为：

- `docs/数据库脚本/zuoye_submit.sql`

不再单独维护 `zuoye_submit_upgrade.sql`。

## 2. 这份脚本现在做什么

`zuoye_submit.sql` 已经合并了“建表”和“升级旧表”两种能力：

1. 如果数据库里还没有 `zuoye_submit` 表，它会直接创建完整表结构
2. 如果数据库里已经有旧版 `zuoye_submit` 表，它会自动补齐缺失字段

当前脚本会确保以下核心字段存在：

- `submit_status`
- `submit_score`
- `submit_remark`
- `check_time`

## 3. 正确执行方式

无论是新库还是旧库，统一执行：

```sql
source docs/数据库脚本/zuoye_submit.sql;
```

如果你是在数据库客户端中手工执行，也可以直接打开这个 SQL 文件运行。

## 4. 这份脚本支撑哪些功能

这份脚本主要支撑下面这条业务链：

1. 学生前台提交作业
2. 后台查看提交记录
3. 教师批改
4. 教师评分
5. 教师填写评语
6. 教师退回重交
7. 前台查看批改结果

如果不执行这份脚本，新前端里的作业提交和批改流程无法完整工作。
